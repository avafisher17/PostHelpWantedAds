# Post Help Wanted Ads
A Stardew Valley/SMAPI mod that allows players to post help wanted ads for in-game items via an in-game menu with quest journal integration.
Item category, player luck, and friendship levels are all factored in when selecting which villager responds to the request. 

***Status: In development — core menu and quest journal functionality complete***

## Features

* Season and item eligibility validation
* Weighted villager selection based on item category
* Save data persistence via SMAPI's data API
* Use of .NET 6 and MonoGame frameworks

## How to Run

1. Install [SMAPI](https://smapi.io/)
2. Clone the repo
3. Copy the PostHelpWantedAds folder into your Stardew Valley mods directory
   (often C:\Program Files (x86)\Steam\steamapps\common\Stardew Valley\Mods)
5. Launch the game via SMAPI

## Technologies Used

* C#
* .NET 6
* MonoGame
* Stardew Modding API, or [SMAPI](https://smapi.io/))
